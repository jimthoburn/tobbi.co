OK the Store

with the team at Barrel Strength Design

OK the Store is a stylish shop with locations in Los Angeles and Silverlake, where you can find beautiful, useful objects for your home, office or the outdoors.

Based on OK the Store’s existing brand, Ben and I designed and built web prototypes for the e-commerce store.

My role also involved writing production-quality HTML, CSS and JavaScript for the user interface, testing for accessibility and browser/device compatibility, and fine-tuning the code for better performance.
