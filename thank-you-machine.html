Thank You Machine

with the teams at Zendesk and One Darnley Road

Thank You Machine is a web app that you can use to send an illustrated card to a colleague, friend or family member.

I teamed up with John & Sam to create a user interface for site, integrate it with a CMS, and deploy it to a scalable web hosting service. We used Netlify functions to generate a downloadable image for each card. And we used stacks of SVG images for the card illustrations, so a visitor can make their card more unique.

The project was completed in a short time frame, going from prototype to launch in a few weeks. Work continued after the first version was released, including a holiday edition and translations into Spanish, Portuguese, French, German, Dutch, Japanese and Korean.

You can read more about the project on the Zendesk web site.
