Thank You Machine

with the teams at Zendesk and One Darnley Road

Thank You Machine is a web app that visitors can use to make an illustrated card for a colleague, friend or family member.

Our engineering lead, John, and I teamed up to create a user interface for the app, integrate it with a CMS, and set it up with scalable hosting. The illustrations were implemented with SVG images that a visitor can mix & match to make their card more unique. And we used Netlify functions to generate a downloadable image for each card.

The project was completed in a short time frame, going from prototype to launch in weeks. Work continued after the first version was released, including a holiday edition and translations into Spanish, Portuguese, French, German, Dutch, Japanese and Korean.

You can read more about the project on the Zendesk web site.
