Fahrenheit Studio

with the team at Fahrenheit Studio

Fahrenheit Studio has been helping companies express their brand since 1995. Their beautiful, desktop-oriented web site needed a redesign.

During the project, I worked alongside Dylan, our designer, to create a responsive portfolio that emphasized Fahrenheit’s stunning project images and would look great on retina displays and big screens. I also found out that Robert is an excellent guitar player.

My role involved facilitating design discussions, getting sign-off from decision-makers, writing production-quality HTML, CSS and JavaScript for the user interface, testing for accessibility and browser/device compatibility, fine-tuning the code for better performance, creating templates with PHP, and setting up hosting with Beanstalk and Media Temple.
